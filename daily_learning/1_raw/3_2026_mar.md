The DataFrame.isin() method in Pandas is a powerful tool for filtering and selecting data within a DataFrame based on specified conditions. ### df.shape
The shape attribute returns a tuple representing the dimensionality of the DataFrame (number of rows, number of columns). It is an attribute, so it does not require parentheses. 

Syntax: df.shape
Returns: A tuple of the form (nrows, ncolumns).
Documentation: pandas.DataFrame.shape documentation 
Example:
python
print("DataFrame shape (rows, columns):")
print(df.shape)

### df.columns
The columns attribute provides the column labels of the DataFrame as a pandas Index object. Like shape, it is an attribute and does not require parentheses. 

Syntax: df.columns
Returns: A pandas Index object containing the column labels.
Documentation: pandas.DataFrame.columns documentation 
Example:
python
print("Column names:")
print(df.columns)
