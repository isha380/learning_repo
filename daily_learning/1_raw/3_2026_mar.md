## Creating Columns in Python (Pandas)

## Ways to Create / Add a Column

---

### Method 1 — Add a Simple New Column

**Syntax:**
```python
df['new_column_name'] = value
```

**What it does:** Adds a brand new column to your existing table.

```python
# Beginner example
df['country'] = 'USA'       # Same value for everyone
print(df)
```
```
      name  age  salary country
0    Alice   25   50000     USA
1      Bob   30   60000     USA
2  Charlie   35   75000     USA
```

```python
# Pro example — adding multiple columns at once
df['country']  = 'USA'
df['currency'] = 'USD'
df['active']   = True
```

---

### Method 2 — Create a Column from Math

**Syntax:**
```python
df['new_column'] = df['existing_column'] * number
```

**What it does:** Does a calculation on an existing column and saves the result as a new column.

```python
# Beginner example
df['bonus'] = df['salary'] * 0.10
print(df[['name', 'salary', 'bonus']])
```
```
      name  salary   bonus
0    Alice   50000  5000.0
1      Bob   60000  6000.0
2  Charlie   75000  7500.0
```

```python
# Pro example — multiple math operations
df['tax']        = df['salary'] * 0.20
df['net_salary'] = df['salary'] - df['tax']
df['total_comp'] = df['net_salary'] + df['bonus']
```

---

### Method 3 — Column with a Condition (If/Else)

**Syntax:**
```python
import numpy as np
df['new_column'] = np.where(condition, value_if_true, value_if_false)
```

**What it does:** Checks each row — if the condition is true, it gives one value, otherwise another.

```python
import numpy as np

# Beginner example
df['is_senior'] = np.where(df['age'] >= 30, 'Yes', 'No')
print(df[['name', 'age', 'is_senior']])
```
```
      name  age is_senior
0    Alice   25        No
1      Bob   30       Yes
2  Charlie   35       Yes
```

```python
# Pro example — multiple conditions using np.select
conditions = [
    df['salary'] < 55000,
    df['salary'].between(55000, 70000),
    df['salary'] > 70000
]
labels = ['Low', 'Medium', 'High']

df['salary_band'] = np.select(conditions, labels, default='Unknown')
```

---

### Method 4 — Column Using `apply()` (Custom Logic)

**Syntax:**
```python
df['new_column'] = df['existing_column'].apply(function)
```

**What it does:** Runs a function on every single row and stores the result as a new column. Use this when your logic is too complex for simple math.

```python
# Beginner example
def label_age(age):
    if age < 28:
        return 'Young'
    else:
        return 'Experienced'

df['age_label'] = df['age'].apply(label_age)
print(df[['name', 'age', 'age_label']])
```
```
      name  age    age_label
0    Alice   25        Young
1      Bob   30  Experienced
2  Charlie   35  Experienced
```

```python
# Pro example — apply across full row using axis=1
def full_profile(row):
    if row['age'] < 28 and row['salary'] < 55000:
        return 'Entry Level'
    elif row['salary'] >= 70000:
        return 'Senior Level'
    return 'Mid Level'

df['profile'] = df.apply(full_profile, axis=1)
```

---

### Method 5 — String Column Operations

**Syntax:**
```python
df['new_column'] = df['text_column'].str.method()
```

**What it does:** Lets you manipulate text columns — uppercase, lowercase, length, splitting, etc.

```python
# Beginner example
df['name_upper']  = df['name'].str.upper()
df['name_length'] = df['name'].str.len()
print(df[['name', 'name_upper', 'name_length']])
```
```
      name name_upper  name_length
0    Alice      ALICE            5
1      Bob        BOB            3
2  Charlie    CHARLIE            7
```

```python
# Pro example
df['first_letter'] = df['name'].str[0]          # First character
df['has_ali']      = df['name'].str.contains('Ali', case=False)
df[['first', 'last']] = df['full_name'].str.split(' ', expand=True)
```

## Quick Reference Cheat Sheet

| Goal                     | What to use                       |
|--------------------------|-----------------------------------|
| Add a fixed value column | `df['col'] = 'value'`             |
| Add a calculated column  | `df['col'] = df['x'] * 2`         |
| Add if/else column       | `np.where(condition, a, b)`       |
| Add with complex logic   | `.apply(your_function)`           |
| Manipulate text column   | `.str.upper()`, `.str.len()`      |
| Multiple conditions      | `np.select(conditions, labels)`   |

# pandas.melt

Unpivot a DataFrame from wide to long format, optionally leaving identifiers set.

This function is useful to reshape a DataFrame into a format where one or more columns are identifier variables (id_vars), while all other columns are considered measured variables (value_vars), and are “unpivoted” to the row axis, leaving just two non-identifier columns, ‘variable’ and ‘value’.

## Python Pandas.melt() Function Syntax
Syntax: pandas.melt(frame, id_vars=None, value_vars=None,var_name=None, value_name='value', col_level=None)

Parameters:

frame : DataFrame

id_vars[tuple, list, or ndarray, optional] : Column(s) to use as identifier variables.

value_vars[tuple, list, or ndarray, optional]: Column(s) to unpivot. If not specified, uses all columns that are not set as id_vars.

var_name[scalar]: Name to use for the ‘variable’ column. If None it uses frame.columns.name or ‘variable’.

value_name[scalar, default ‘value’]: Name to use for the ‘value’ column.

col_level[int or string, optional]: If columns are a MultiIndex then use this level to melt.

# Pandas dataframe.groupby() Method
- Pandas groupby() function is a powerful tool used to split a DataFrame into groups based on one or more columns, allowing for efficient data analysis and aggregation. 

- It follows a "split-apply-combine" strategy, where data is divided into groups, a function is applied to each group, and the results are combined into a new DataFrame.

### DataFrame.groupby(by=None, axis=0, level=None, as_index=True, sort=True, group_keys=True, observed=False, dropna=True)

Parameters :

by: Required parameter to specify the column(s) to group by.
axis: Optional, specifies the axis to group by (default is 0 for rows).
level: Optional, used for grouping by a certain level in a MultiIndex.
as_index: Optional, whether to use the group labels as the index (default is True).
sort: Optional, whether to sort the group keys (default is True).
group_keys: Optional, whether to add the group keys to the index (default is True).
dropna: Optional, whether to include rows/columns with NULL values (default is True).
